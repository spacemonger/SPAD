# Single Photon Avalanche Diode (SPAD)

## Overview
The Single Photon Avalanche Diode (SPAD) project aims to develop and simulate the functionalities of a SPAD using advanced semiconductor technologies. This repository contains all relevant materials, including models, simulations, and documentation for easy reference and usage.

## 65 nm TSMC Process
This project utilizes the 65 nm process technology from TSMC, which offers advanced performance characteristics crucial for SPAD functionality. This process allows for reduced power consumption and smaller device dimensions, enhancing the performance of the SPAD in various applications.

## Verilog-A Behavioral Model
The SPAD project includes a comprehensive Verilog-A behavioral model that accurately represents the dynamics and operation of the Single Photon Avalanche Diode. This model can be used for simulations and can be integrated into larger circuit designs.

## Planned Additions
Future plans for the project include:
- **Sentaurus TCAD Integration:** Incorporating Sentaurus TCAD simulations for deeper analysis of the device behavior and performance under various conditions.
- **Layout Pictures:** Adding layout visualizations to demonstrate the physical arrangement of the SPAD and its components.
